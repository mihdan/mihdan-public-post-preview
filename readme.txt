== Installation ==

Внимание! У плагина нет страницы с настройками, его надо просто включить и он будет работать как швейцарские часы.

**Автоматическая установка из админки WordPress**

1. Перейдите по пути `Плагины` -> `Добавить новый` в панели администрирования
2. Вбейте в поисковую строку `Mihdan: Public Post Preview`
3. Щёлкните кнопку `Установить` на найденном плагине
4. Щёлкните кнопку `Активировать`

**Ручная установка по FTP**

1. Закачайте папку `mihdan-public-post-preview` в директорию `/wp-content/plugins/ directory`
2. Активируйте плагин по пути `Плагины` в панели администрирования

**Ручная установка из ZIP**

1. Закачайте архив по пути `Плагины` -> `Добавить новый` в панели администрирования
2. Щёлкните кнопку `Активировать`
